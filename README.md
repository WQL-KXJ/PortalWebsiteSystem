## 企业门户网站，基于Springboot和layui前后端分离
### 使用技术：
#### 后端框架：SpringBoot，Mybatisplus
#### 数据库：MySQL,redis
#### 前端框架：Layui
#### 权限框架：shiro
#### 网页模板引擎：thymeleaf
